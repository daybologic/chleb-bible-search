Welcome to Chleb Bible Search __VERSION__

This is a microservice for querying and searching the Bible. You can use the following endpoints directly, or explore the Swagger API documentation.

  • Search for verses (e.g. /1/search?term=love)
  • Verse of the Day (/2/votd)
  • Random Verse (/2/random?testament=any)
  • Lookup a Verse (e.g. /1/lookup/john/3/16)
  • Bible Info (/1/info)
  • Ping (/1/ping)
  • Service Version (/1/version)
  • Service Uptime (/1/uptime)
